# To-Do List აპლიკაცია

To-Do List აპლიკაცია React-ით შექმნილი.

## ფუნქციები

- ✅ ორი სვეტი: შესასრულებელი და შესრულებული დავალებები
- ✅ ახალი დავალების დამატება
- ✅ დავალების დასრულება (შესასრულებელიდან შესრულებულში გადატანა)
- ✅ შესრულებული დავალების წაშლა
- ✅ შესრულებული დავალების უკან დაბრუნება შესასრულებელში
- ✅ თანამედროვე და ლამაზი დიზაინი
- ✅ რესპონსიული (მობილურზე მოსახერხებელი)

## გაშვება

1. დააინსტალირეთ დამოკიდებულებები:
\`\`\`bash
npm install
\`\`\`

2. გაუშვით აპლიკაცია:
\`\`\`bash
npm start
\`\`\`

3. გახსენით ბრაუზერში: [http://localhost:3000](http://localhost:3000)

## გამოყენება

1. **დავალების დამატება**: შეიყვანეთ დავალების ტექსტი ველში და დააჭირეთ "დამატება" ღილაკს ან Enter-ს
2. **დავალების დასრულება**: დააჭირეთ "დასრულება" ღილაკს შესასრულებელ დავალებაზე
3. **დავალების წაშლა**: დააჭირეთ "წაშლა" ღილაკს შესრულებულ დავალებაზე
4. **დავალების უკან დაბრუნება**: დააჭირეთ "უკან" ღილაკს შესრულებულ დავალებაზე

## ტექნოლოგიები

- React 18
- React Hooks (useState)
- CSS3 (Grid, Flexbox, Animations)
- Modern ES6+ JavaScript
