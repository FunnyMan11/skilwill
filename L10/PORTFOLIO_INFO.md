# პორტფოლიო პროექტი

## მიმოხილვა
ეს არის მარტივი პორტფოლიო ვებ გვერდი, შექმნილი React-ის გამოყენებით.

## შექმნის პროცესი
1. პროექტის ინიციალიზაცია `create-react-app` პაკეტის და `npx` ბრძანების გამოყენებით
2. საწყისი გვერდის მოდიფიკაცია პორტფოლიო დიზაინში
3. პროფილის სურათისა და ტექსტური ინფორმაციის დამატება

## ფუნქციონალი
- **პროფილის სექცია**: სურათი, სახელი და პოზიცია
- **შესახებ სექცია**: მოკლე აღწერა დეველოპერის შესახებ
- **უნარები**: Frontend, Backend და Tools განყოფილებები
- **კონტაქტი**: ელ.ფოსტა და ტელეფონის ნომერი

## დიზაინი
- თანამედროვე gradient ფონი (იისფერი-მეწამული)
- Glass morphism ეფექტები
- Responsive დიზაინი (მორგებული მობილურზე)
- Hover ანიმაციები უნარების card-ებზე
- Fade-in ანიმაცია პროფილის სექციისთვის

## გაშვება
```bash
cd L10
npm start
```

აპლიკაცია გაიხსნება ბრაუზერში მისამართზე: http://localhost:3000

## ტექნოლოგიები
- React 19.2.0
- CSS3 (Flexbox, Grid, Animations)
- Modern JavaScript (ES6+)
